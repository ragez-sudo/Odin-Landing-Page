# Odin Landing Page

This holds the html and css files of the Odin Project: Landing Page found at the end of the Fexbox section in Foundations.

It is a test of HTML, CSS, and Flexbox usage skills developed so far throughout the course and also utilizes Git usage to keep create, link to, and keep this repository updated.

The only guidance explicitly offered in completing this project are two reference images: one serving as a model of the desired final design and the other serving as a color and font index that the final design utilizes.

All work to create the webpage produced by the HTML in this reposistory was done solely by myself and the various sources I may have referenced during the completion of this project.
